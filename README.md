# Countries App
Приложение на React + TypeScript, которое отображает список стран с возможностью поиска, фильтрации по региону и просмотра детальной информации о каждой стране.

# Как запустить
1. `git clone <repo-url>`

2. `cd <project-name>`
3. `npm install`
4. `npm run dev`
5. Откройте http://localhost:5173
